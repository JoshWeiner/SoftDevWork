Buttons

Coloring options: primary, secondary, success, alert, warning

Button Sizing:

tiny, small, large, expanded

Changes the button's size based on the class of button that is specified. Expanded will change the width of the button to fill the entire container.


Other Options:

hollow, clear

Hollow creates a button without a background but with a border.
Clear creates a button without a background nor a border.
Disabled creates a button that looks like it is disabled; the disabled class must be added to actually disable it.
